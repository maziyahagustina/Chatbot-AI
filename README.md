# AI Chatbot System (Healthcare Use Case)

## 📌 Problem
Manual handling of internal user queries was inefficient and time-consuming, leading to slow response times and inconsistent information delivery.

## 💡 Solution
Developed an AI-powered chatbot using Large Language Models (LLM) integrated with Retrieval-Augmented Generation (RAG) and LangChain.  
The system connects to a PostgreSQL database to retrieve relevant data and generate accurate, context-aware responses.

## ⚙️ Tech Stack
- Python  
- SQL (PostgreSQL)  
- Pandas  
- LangChain  
- LLM (OpenAI / similar)  
- RAG (Retrieval-Augmented Generation)  

## 🚀 Result
- Improved response relevance and accuracy  
- Reduced manual query handling by approximately 50%  
- Enhanced efficiency in accessing internal information
